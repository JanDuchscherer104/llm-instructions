# LLM-Instructions

This repository contains agent instructions from various projects.

- [**traenslenzor**](https://github.com/luroess/traenslenzor): Advanced document translation system leveraging local LLMs and modular tools for high-quality translations (WS25/26).
- [**aria-nbv**](https://github.com/JanDuchscherer104/NBV): RRI-based NBV leveraging egocentric foundation models (WS25/26).
- [**ama_ws25**](https://github.com/punsii/ama_ws25): Applied Multivariate Data Analysis (WS25/26).
